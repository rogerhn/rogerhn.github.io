Galaxies

The universe of No Man's Sky has many galaxies, each with several star systems. All star systems have planets, sometimes with moons, and a single space station.

Average start distance to the galaxy center: 180.000 Light Years

Spoiler

According to game files, the procedural engine can generate up to 256 distinct-looking galaxies (using an 8=bit seed); galaxies after the number 256 will start looking like previously generated ones.

The only difference between the galaxies is its core color.
